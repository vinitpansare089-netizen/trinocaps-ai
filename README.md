# TrinoCaps AI 🎓

An AI-powered assistant to simplify university rules and procedures.

## Features
- Semantic search using Sentence Transformers
- Top relevant rule retrieval
- Category detection (Attendance, Hostel, Exams, etc.)
- Explainable AI (shows why an answer is selected)
- Smart question suggestions
- Analytics dashboard

## Tech Stack
- Python
- Streamlit
- SentenceTransformers
- Scikit-learn

## Use Case
Students can quickly get answers about university rules without reading long PDFs.
